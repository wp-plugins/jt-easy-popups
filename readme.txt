== Description ==

* 간단한 이미지 팝업을 쉽게 설정하여 사용할 수 있습니다.
* 여러개의 팝업을 동시에 사용할 수 있습니다.
* 기간을 설정하여 팝업을 개시할 수 있습니다.

== Installation ==

1. 다운로드 받은 플러그인을 FTP를 통해 'Plugins' 폴더로 업로드 하거나, 
   워드프레스 관리자-> 플러그인 탭 에서 'jt-easy-popups' 검색 후 설치 
2. 'JT Easy Popups' 플러그인 활성화
