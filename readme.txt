== Description ==

※ 2014.05.28 - 플러그인 수정중 입니다. 지금은 플러그인을 이용할 수 없습니다.

※ 기존에 1.0.0 버전의 플러그인이 설치되어 있다면, 업데이트가 아닌 기존의 플러그인을 완전히 삭제한 후 1.0.1 버전을 새로 설치해야 합니다.

* 간단한 이미지 팝업을 쉽게 설정하여 사용할 수 있습니다.
* 여러개의 팝업을 동시에 사용할 수 있습니다.
* 기간을 설정하여 팝업을 개시할 수 있습니다.

== Installation ==

1. 다운로드 받은 플러그인을 FTP를 통해 'Plugins' 폴더로 업로드 하거나, 
   워드프레스 관리자-> 플러그인 탭 에서 'jt-easy-popups' 검색 후 설치 
2. 'JT Easy Popups' 플러그인 활성화

== Changelog ==

= 1.0.0 =
* 플러그인 등록

= 1.0.1 =
* 팝업 스타일 추가.
* 팝업의 그림자를 선택적으로 사용할 수 있습니다.
* 팝업의 둥근 모서리를 4가지 옵션으로 선택할 수 있습니다.
* 팝업의 순서로 생성되던 쿠키를 팝업 개별로 생성하도록 수정.
* '오늘 하루 보지 않음'의 폰트와 위치 수정.
* 팝업의 개시기간의 기본값이 당일날짜로 수정.
* 플러그인 비주얼과 스크린샷 추가.
